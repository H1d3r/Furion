【什么是 Furion？】
Furion 是一个应用程序框架，您可以将它集成到任何 .NET/C# 应用程序中。

【核心特性】
√  模块化架构设计
√  内置企业级开发最佳实践
√  零依赖设计
√  支持 .NET 5/6/7/8/9/10 及跨平台部署
√  开箱即用的常用工具集（日志/缓存/数据库等）

【快速开始】
1. 访问官方文档：https://furion.net 
2. 查看 Gitee 仓库：https://gitee.com/dotnetchina/Furion 
3. 查看 Github 仓库（国际镜像）： https://github.com/monksoul/Furion
4. 示例项目：在解决方案中搜索 "samples/Furion.sln" 开始体验
